# EvilQR

EvilQR es una herramienta escrita en Bash que te permite generar fácilmente códigos QR a partir de enlaces.

## Uso

Para generar un código QR, sigue las instrucciones:

1- git clone https://github.com/abund4nt/EvilQR <br>
2- cd EvilQR <br>
3- chmod +x ./setup.sh <br>
4- ./setup.sh <br>
5- ./EvilQR <br>

Una vez que hayas proporcionado la información necesaria, EvilQR generará el código QR en un archivo con extensión .png en tu directorio actual. El archivo se nombrará automáticamente según el contenido del código QR.

Ten en cuenta que el mal uso de EvilQR puede tener consecuencias legales y éticas. NO me hago responsable del mal uso de la misma.
